# IP Logger

A web application that detects and logs IP addresses.

## Features
- Detects multiple IP address types
- Simple one-click interface
- Real-time IP logging

## Setup
```bash
npm install
npm start
```

## Usage
1. Start the server: `npm start`
2. Access the application in your browser
3. Click the button to log IP addresses

## Technical Details
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- IP Detection: WebRTC and external APIs

## Disclaimer
This project is for educational purposes only.
